Detailed Description

link的算法： Step1： 将84个字母打乱（包含BCDEFGHIJ各八个，A12个） Step2： 如果Step1之后最后一个字母为A，则在A后面插入X； Step3： 插入12个X（如果经过Step则插入11个X），且不在第一个（call repeat方法） Step4： 如果Step3之后，有AX相连，则中间插入一个AX Step5： 将剩余数量的AX插入其中，且不在首尾，且不相连（call nonrepeat方法）

Random算法： Step1：将96个字母打乱（包含12个A，12个X，BCDEFGHIJ各8个）， Step2: 检查当前情况下，是否有AX相连，如果有的话，插入AX于其中 Step3：检查第一个和最后一个是否分别为X和A，如果是的话，插入AX到最前与最后 Step4: 将剩余AX插入，注意检查AX不能放入之前的AX中间
